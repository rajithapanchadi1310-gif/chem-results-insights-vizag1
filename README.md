# 📊 Vizag-1 Chemistry Results Dashboard

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing **section-wise chemistry results** for the Vizag-1 branch.  
It helps in identifying performance trends, comparing sections, and improving academic insights.

---

## 🎯 Objectives
- Analyze student performance section-wise
- Identify high and low performing sections
- Track trends in marks distribution
- Support data-driven academic decisions

---

## 🛠️ Tools Used
- Power BI (.pbix)
- Excel (data source)

---

## 📂 Project Structure
```
├── data/          # Dataset files
├── reports/       # Power BI file (.pbix)
├── images/        # Dashboard screenshots
└── README.md
```

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Explore the dashboard visuals  

---

## 📈 Key Insights
- Section-wise performance comparison  
- Top and low scoring section in a pie-chart 
- Trends in marks distribution
